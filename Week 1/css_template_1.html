You can add a box to something quite easily. For example, you can define your own special class of an XHTML tag (<div> in this case) and give it a class attribute ("tarzan" in this case). Put this (or a similar) style definition in your style sheet.

div.tarzan { border: solid thick black; background-color: #FFCCCC; padding: 2em}

To apply this style to a division in the <BODY> of your text, you need to give the opening tag some class. You'll need to change the HTML to:

<div class="tarzan">
    ...
    </div>
